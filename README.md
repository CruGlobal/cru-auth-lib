# CruAuthLib
Collection of common auth models for use in shared API authentication.

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'cru-auth-lib'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install cru-auth-lib
```
